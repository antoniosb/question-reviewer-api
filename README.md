# API Question Reviewer

## Version
 * Ruby (2.5.1)

## Dependencies
 * Bundler
    ```
    gem install bundler
 * Rails
    ```
    gem install rails
 * Install Gems
    ```
    bundle install

## Database creation
    rake db:migration
    
## Database initialization
    rake db:seed

## Run tests
    rake
    
## Init dev server
    rails s
